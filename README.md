###gearman_worker_framewrok

####intro

    使用c语言更简单的创建一个gearman worker端
    
    
####Install

    cd gearman_worker_framework
    cmake .
    make 
    
    ./bin/gearman_worker_test -c ./bin/test.ini
    
    
####Files

    | - bin    (编译后的执行文件)
    | - framework (框架文件)
    | - lib       (第三方库)    
    | - src       （源码）    
    | - php_test   (php 测试脚本)
    | - CmakeLists.txt
